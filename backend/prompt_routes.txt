Buatkan file routing berikut di src/routes/:

auth.routes.js
- POST /register
- POST /login
- POST /logout
- POST /forgot-password
- POST /reset-password

users.routes.js (admin only)
- GET /
- GET /:id
- PATCH /:id
- DELETE /:id

papers.routes.js
- POST /
- GET /
- GET /:id
- GET /:id/download
- PATCH /:id
- POST /:id/version
- DELETE /:id

artifacts.routes.js
- POST /:paperId/artifacts
- GET /:paperId/artifacts
- GET /:id
- GET /:id/download
- PATCH /:id
- DELETE /:id

readingLists.routes.js
- POST /
- GET /me
- GET /:id
- PATCH /:id
- DELETE /:id
- POST /:id/items
- DELETE /:id/items/:paperId

search.routes.js
- GET /

stats.routes.js
- GET /papers/top-downloads
- GET /papers/top-views

Pastikan semua routes import controller + authRequired + RBAC sesuai aturan.
Hubungkan semuanya di src/routes/index.js dengan prefix /api/v1/.
