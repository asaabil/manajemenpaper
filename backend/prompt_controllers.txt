src/services/readingList.service.js
src/services/search.service.js
src/services/stats.service.js
src/services/mail.service.mock.js

Isi setiap controller dengan CRUD sesuai requirement:
- Auth: register, login, logout, forgotPassword, resetPassword
- Users: admin only list/detail/update/delete
- Papers: upload, list, detail (+viewCount), download (+downloadCount), update metadata, add version, delete
- Artifacts: upload, list by paper, detail, download (+downloadCount), update, delete
- ReadingLists: create, getMy, getById, update, delete, addPaper, removePaper
- Search: full-text search (pakai MongoDB Atlas Search pipeline builder)
- Stats: top downloads, top views
