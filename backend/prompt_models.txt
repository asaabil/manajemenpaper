Buatkan file models berikut di src/models/:

User.js
- name, email (unique, lowercase), passwordHash, role (enum: admin, dosen, mahasiswa), affiliation, timestamps

Paper.js
- owner (User ref), title, abstract, authors[], institution, keywords[], publicationDate, categories[]
- file: { path, filename, mimetype, size }
- versions[]: { path, filename, mimetype, size, note, createdAt }
- viewCount, downloadCount, isPublic, timestamps

Artifact.js
- paper (ref), name, description, type (enum: dataset, diagram, code, document, other)
- technology[], language[], methodology[]
- file: { path, filename, mimetype, size }
- externalRepoUrl, downloadCount, isPublic, timestamps

ReadingList.js
- owner (User ref), name, description, isPublic
- items[]: { paper (ref), addedAt }
- timestamps

Gunakan Mongoose schema + export default mongoose.model.
